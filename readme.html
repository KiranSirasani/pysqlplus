Library

The Py Tools library contains Python built applications that can be used to automate menial tasks or increase efficiency.

Front-end tools

The Py Tools front-end tools can be used by anyone to increase their productivity and avoid repetative work. Available tools include;

Excel2SQL Bucket and AutoBucket

The Excel2SQL tool can be used to automatically upload one or more files to SQL Server, click here for documentation.

The Bucket tool is not complete, click here for documentation.


Back-end libraries

The back-end libraries are easy to use but generalised to be used for many different use-cases. The front-end tools package the back-end libraries up into simple tools, but at times more value can be taken from using the back-end libraries directly.

The back-end libraries are split into the following;

data bucket prompt

The data module covers data import/export operations and SQL server interaction. Click here for documentation.

The (incomplete) bucket module covers data bucketing, containing both manual operations and AutoBucket algorithms. Click here for documentation.

The (incomplete) prompt module covers user interaction and front-end tool use. Click here for documentation.
